# Proyecto_Final-Alejandro-Islas: Je ne sais quac
El inicio del desarrollo de una página web para aprender el idioma francés.

## Requisitos Previos

- [Python](https://www.python.org/) (versión X.X.X)
- [pip](https://pip.pypa.io/en/stable/)

## Instalación

1. Clona el repositorio:

    ```bash
    git clone https://github.com/alexisles06/Proyecto_Final-Alejandro-Islas.git
    cd Proyecto_Final-Alejandro-Islas
    ```

2. Crea y activa un entorno virtual (opcional pero recomendado):

    ```bash
    python -m venv venv
    source venv/bin/activate  # Para sistemas basados en Unix
    # o
    .\venv\Scripts\activate  # Para sistemas basados en Windows
    ```

3. Instala las dependencias:

    ```bash
    pip install -r requirements.txt
    ```

## Configuración

1. Copia el archivo de configuración de ejemplo:

    ```bash
    cp config.example.py config.py
    ```

2. Edita el archivo `config.py` según tus necesidades.

## Ejecución

Ejecuta la aplicación con el siguiente comando:

```bash
python app.py
